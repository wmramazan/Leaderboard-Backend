# Leaderboard-Backend
A simple backend application for leaderboard.

## Installation

```bash
$ npm install
```

## Start Docker container for MongoDB
```bash
$ docker-compose up
```

## Stop Docker container for MongoDB
```bash
$ docker-compose down
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
